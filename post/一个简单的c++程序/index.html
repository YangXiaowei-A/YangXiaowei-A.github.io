一个简单的C++程序

int main()
{
	return 0;
}

几点解释：

  • 每个c++程序都包含一个或多个函数，其中一个必须命名为main。
  • 一个函数的定义包含四部分：返回类型、函数名、形参列表和函数体。
  • main函数的返回类型必须为int，即整数类型。
  • 在大多数系统中，main的返回值被用来指示状态。返回值0表明成功，非0的返回值的含义由系统定义，通常用来指出错误类型。
